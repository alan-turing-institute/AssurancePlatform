Template Checklist

Before publishing a module, ensure:

  • [ ] All frontmatter fields are completed accurately
  • [ ] Learning objectives are clear and measurable
  • [ ] Content flows logically and progressively
  • [ ] Examples and scenarios are relevant and clear
  • [ ] All cross-references and links work correctly
  • [ ] Images have alt text and proper attribution
  • [ ] Callouts and formatting enhance readability
  • [ ] Practice exercise is achievable and relevant
  • [ ] Assessment aligns with learning objectives
  • [ ] Resources provide value and are accessible
  • [ ] Module has been reviewed for accuracy
  • [ ] Estimated time is realistic
  • [ ] Grammar and spelling have been checked
  • [ ] Component imports use absolute paths
