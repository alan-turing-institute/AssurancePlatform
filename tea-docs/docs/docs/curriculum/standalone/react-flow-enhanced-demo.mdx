## Usage Instructions

### Creating Nodes
1. **Double-click** anywhere on the empty canvas
2. A **Node Type Selector** dialog will appear
3. Choose from: Goal, Strategy, Property Claim, Evidence, or Context
4. Fill in the node details (optional)
5. Click "Create" to add the node

### Editing Nodes
1. **Right-click** on any node
2. Select "Edit" from the context menu
3. Modify the node properties in the dialog
4. Click "Save" to apply changes

### Managing Connections
1. Drag from a node's handle (+ decorator) to another node
2. Right-click on connections to modify or delete them
3. Use the context menu to change connection styles

### Layout Options
- **Reveal All**: Show all hidden nodes at once
- **Reset**: Return to initial view with only top-level nodes
- **Auto Layout**: Automatically arrange nodes with proper spacing
- **Fit View**: Zoom to fit all visible nodes
