# Technical Documentation

This section provides technical documentation for developers who want to contribute to the TEA Platform or deploy their own instance.

## Tech Stack

The TEA Platform is built with modern web technologies. Here's an overview of the key technologies and links to their official documentation:

| Technology | Purpose | Documentation |
|------------|---------|---------------|
| **Next.js 15** | React framework with App Router | [nextjs.org/docs](https://nextjs.org/docs) |
| **React 19** | UI component library | [react.dev](https://react.dev) |
| **TypeScript** | Type-safe JavaScript | [typescriptlang.org/docs](https://www.typescriptlang.org/docs/) |
| **Prisma 7** | Database ORM for PostgreSQL | [prisma.io/docs](https://www.prisma.io/docs) |
| **PostgreSQL** | Relational database | [postgresql.org/docs](https://www.postgresql.org/docs/) |
| **ReactFlow** | Node-based diagram editor | [reactflow.dev](https://reactflow.dev) |
| **Tailwind CSS** | Utility-first CSS framework | [tailwindcss.com/docs](https://tailwindcss.com/docs) |
| **shadcn/ui** | Accessible UI components | [ui.shadcn.com](https://ui.shadcn.com) |
| **NextAuth.js** | Authentication library | [authjs.dev](https://authjs.dev) |
| **Biome** | Linting and formatting | [biomejs.dev](https://biomejs.dev) |
