**Instructions:** Copy this template when creating a new module, fill in all sections, and delete instructional comments (lines starting with `<!-- `).

  - [learning-modules]

authors:
  - [Author handle/name]

## Component Usage Examples

### Using Card Components

```markdown
import { Cards } from 'nextra/components'

<Cards>
  <Cards.Card
    title="Card Title"
    href="./relative-path"
  />
  <Cards.Card
    title="Another Card"
    href="./another-path"
  />
</Cards>
```

### Using Callouts

```markdown
import { Callout } from 'nextra/components'

<Callout>
Standard informational note
</Callout>

<Callout type="info">
**Pro Tip**

Helpful advice or best practice
</Callout>

<Callout>
**Key Concept**

Important concept or definition
</Callout>

<Callout type="info">
**Well Done!**

Positive reinforcement or achievement
</Callout>

<Callout type="warning">
**Caution**

Warning about common mistakes
</Callout>

<Callout type="error">
**Important**

Critical information that must not be missed
</Callout>
```

### Using Collapsible Sections

```markdown
<details>
<summary>Click to expand</summary>

Hidden content goes here. Can include any markdown formatting.

</details>
```

### Using Tabs

```markdown
import { Tabs } from 'nextra/components'

<Tabs items={['Option 1', 'Option 2']}>
  <Tabs.Tab>
    Content for option 1
  </Tabs.Tab>
  <Tabs.Tab>
    Content for option 2
  </Tabs.Tab>
</Tabs>
```

## Template Checklist

Before publishing a module, ensure:

* \[ ] All frontmatter fields are completed accurately
* \[ ] Learning objectives are clear and measurable
* \[ ] Content flows logically and progressively
* \[ ] Examples and scenarios are relevant and clear
* \[ ] All cross-references and links work correctly
* \[ ] Images have alt text and proper attribution
* \[ ] Callouts and formatting enhance readability
* \[ ] Practice exercise is achievable and relevant
* \[ ] Assessment aligns with learning objectives
* \[ ] Resources provide value and are accessible
* \[ ] Module has been reviewed for accuracy
* \[ ] Estimated time is realistic
* \[ ] Grammar and spelling have been checked
* \[ ] Component imports use absolute paths
