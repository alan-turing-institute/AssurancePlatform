Welcome to the Trustworthy and Ethical Assurance (TEA) documentation. These pages contain all you need to know about our collaborative platform for building and managing assurance cases.

## Documentation Structure

This documentation is organised into several sections:

### TEA Curriculum

Step-by-step learning paths for understanding and applying trustworthy and ethical assurance:

- **TEA Trainee**: Introduction to assurance case fundamentals
- **TEA Specialist**: Intermediate techniques and methodologies
- **TEA Expert**: Advanced topics and real-world applications

### Technical Documentation

Resources for developers and system administrators:

- Setup and installation guides
- Frontend and backend reference
- CI/CD pipeline documentation
- API documentation for integrating with the TEA Platform

### Community Resources

Guidelines for contributing, reporting issues, and engaging with the TEA community.
